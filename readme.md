# BEST_CRUD_PROJECT

- 리포지터리 설명: 우아한 테크코스 프리코스 4-5주차 미션으로 진행한 프로젝트입니다.
- 기획 노션: [링크](https://www.notion.so/CRUD-2a3950a6038b807a87fbccf384c5f990)

## 선정 계기

"이 프로젝트는 '어떻게'보다 '왜'를 깊이 고민하기 위한 의식적인 훈련입니다.

그동안 여러 프로젝트에서 마감 기한에 맞춰 기능을 구현하는 데 급급했습니다. 하지만 나중에 코드를 유지보수하며 "이 코드는 왜 이렇게 작성했을까?"라는 아쉬움과 마주하곤 했습니다.

따라서 이번에는 기능이 단순한 MVP(게시판)를 통해, 코드의 '질'을 높이고 확장성을 고민하는 데 집중하고자 합니다. 같은 기획을 두고 서로의 코드를 리뷰하는 프리코스 방식이, 이런 저의 목표를 달성하는 데 최적이라고 생각하여 참여했습니다."

## 기술 스택, 선정 이유

- Next.js
  - SSR에 대한 이해도를 더 높이고, Hydration 방식을 더 고민하고 사용하기 위해 선정했습니다.
- Tailwind-CSS
  - 디자인 토큰을 쉽게 관리할 수 있고, 토큰 자동화도 제공하기 때문에 디자인 시스템을 설계하기 가장 적절하다고 생각하여 선정했습니다.
